
# Glasgow University Recipes - GUR

In this project we have used the whoosh search engine. To set this up you should run:

``` python3 manage.py rebuild_index ```

After running the population script.

If this is not done correctly the search bar will always return no results.

We also used Select2 -- an open source, jQuery based replacement for select boxes -- to create the tagging style used for ingredients. Source: https://github.com/codingjoe/django-select2?tab=readme-ov-file
